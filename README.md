# Paradigm Pet Professionals – D279 UI Design

This repo contains my full HTML/CSS prototype for the Paradigm Pet Professionals website, created for WGU course D279 (User Interface Design).

## Contents

- `index.html` – Home (overview, adoption resources, call to consultation form)
- `dogs.html` – Dog Owners (diet, exercise, grooming, vaccines)
- `cats.html` – Cat Owners (nutrition by life stage)
- `small-animals.html` – Small-Animal Owners (housing & habitat, enrichment resources)
- `contact.html` – Consultation request form (name, phone, email, time zone, pet info)

Supporting files:

- `css/styles.css` – Site-wide styling
- `images/` – Logo and hero imagery
- `docs/D279_Performance_Assessment.pdf` – Written performance assessment (Part I)
